## **Project Overview**

A feature-rich blogging platform demonstrating proficiency in full-stack development, database design, and RESTful API architecture. The application includes user authentication with session management, a sophisticated tagging system using many-to-many relationships, full-text search across posts, and a nested comment system.

Key technical implementations include PostgreSQL database optimization with indexing, server-side rendering with Jinja2 templates, client-side interactivity with vanilla JavaScript, and responsive design following mobile-first principles. Security features include bcrypt password hashing, parameterized SQL queries to prevent injection, and CSRF protection.

**Tech Stack:** Python, Flask, PostgreSQL, JavaScript, HTML5, CSS3, Jinja2
